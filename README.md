# 00_CORE

## EN

Install once:

1. Install Git: https://git-scm.com/download/win
2. Install the Git LFS app: https://git-lfs.com/
3. In the folder where the library should be stored, right click and choose `Open Git Bash here`
4. Run one-time setup and clone:

```bash
git lfs install
git clone https://github.com/multipapl/00_CORE.git
```

To update later:

1. Open the `00_CORE` folder
2. Right click and choose `Open Git Bash here`
3. Run:

```bash
git pull
```

## UA

Встановити один раз:

1. Встановіть Git: https://git-scm.com/download/win
2. Встановіть програму Git LFS: https://git-lfs.com/
3. У папці, де має лежати бібліотека, натисніть правою кнопкою миші і виберіть `Open Git Bash here`
4. Виконайте одноразове налаштування і клонування:

```bash
git lfs install
git clone https://github.com/multipapl/00_CORE.git
```

Щоб потім оновлювати:

1. Відкрийте папку `00_CORE`
2. Натисніть правою кнопкою миші і виберіть `Open Git Bash here`
3. Виконайте:

```bash
git pull
```
