# 00_CORE

## EN

### Recommended: Anchorpoint

Best option if you do not use Git.

1. Install Anchorpoint: https://www.anchorpoint.app/
2. Sign in to Anchorpoint
3. In Anchorpoint create a new project
4. Choose `Connect existing repository via https`
5. Paste this URL:

```text
https://github.com/multipapl/00_CORE.git
```

6. Choose the folder where the library should be stored
7. Start the download and wait until it finishes

To update later:

1. Open the project in Anchorpoint
2. Click `Pull`

### Alternative: Git Bash

Use this if you do not want Anchorpoint.

1. Install Git: https://git-scm.com/download/win
2. Install Git LFS: https://git-lfs.com/
3. In the folder where the library should be stored, right click and choose `Open Git Bash here`
4. Run:

```bash
git lfs install
git clone https://github.com/multipapl/00_CORE.git
```

To update later:

1. Open the `00_CORE` folder
2. Right click and choose `Open Git Bash here`
3. Run:

```bash
git pull
```

## UA

### Рекомендовано: Anchorpoint

Найпростіший варіант, якщо ви не користуєтесь Git.

1. Встановіть Anchorpoint: https://www.anchorpoint.app/
2. Увійдіть в Anchorpoint
3. У Anchorpoint створіть новий проект
4. Виберіть `Connect existing repository via https`
5. Вставте це посилання:

```text
https://github.com/multipapl/00_CORE.git
```

6. Виберіть папку, де має лежати бібліотека
7. Запустіть завантаження і дочекайтесь завершення

Щоб потім оновлювати:

1. Відкрийте проект в Anchorpoint
2. Натисніть `Pull`

### Альтернатива: Git Bash

Використовуйте цей варіант, якщо не хочете Anchorpoint.

1. Встановіть Git: https://git-scm.com/download/win
2. Встановіть Git LFS: https://git-lfs.com/
3. У папці, де має лежати бібліотека, натисніть правою кнопкою миші і виберіть `Open Git Bash here`
4. Виконайте:

```bash
git lfs install
git clone https://github.com/multipapl/00_CORE.git
```

Щоб потім оновлювати:

1. Відкрийте папку `00_CORE`
2. Натисніть правою кнопкою миші і виберіть `Open Git Bash here`
3. Виконайте:

```bash
git pull
```
